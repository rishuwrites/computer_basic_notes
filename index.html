Keyboard – Basic Knowledge
An input device used to type letters, numbers and give commands
🔑 Main Keys and Their Work
Alphabet Keys (A–Z) → Type letters and words
Number Keys (0–9) → Type numbers
Enter → New line / confirm command
Backspace → Delete letter to the left
Spacebar → Space between words
Shift → Capital letter or symbol (Shift + a = A)
Caps Lock → Capital letters ON / OFF
Tab → Move cursor forward
Esc (Escape) → Cancel current action
Ctrl (Control) → Used with shortcuts
Windows Key ⊞ → Open Start menu

Common Beginner Mistakes
Avoid these — they confuse almost every new computer user
✖Thinking Minimize = Close — it does not close the app!
✖Closing a file without saving first (always Ctrl + S)
✖Using Shift + Delete by accident — permanently deletes!
✖Panicking when a window disappears — it is just minimized
✅ Remember: Minimized apps are safe. Look for them in the Taskbar at the bottom of the screen.

How to Change Wallpaper & Lock Screen
Personalise the look of your Windows desktop
  1. Right-click on the Desktop (empty area)
  2. Click Personalize from the menu
  3. Choose Background → select your wallpaper image
  4. Choose Lock Screen → change the lock screen image

How to Create, Rename & Delete Folders
Organise your files using folders
📂 Create Folder
  • Right-click Desktop → New → Folder
  • Shortcut: Ctrl+Shift+N
✏️ Rename Folder
  • Right-click the folder → Rename
  • Select folder → press F2
🗑️ Delete Folder (Temporary)
  • Select folder → press Delete
  • Goes to Recycle Bin (recoverable)
♻️ Recover Deleted Folder
  • Open Recycle Bin
  • Right-click the file
  • Click Restore
⚠️ Permanent Delete: Select folder → Shift+Delete — this action cannot be undone. The file will NOT go to the Recycle Bin.

What is Notepad?
A basic text editor built into Microsoft Windows
💼 Uses of Notepad
  • Write simple text and quick notes
  • Create and save .txt files
  • Write HTML / CSS / JavaScript code
  • Write batch files (.bat) for Windows automation
  • Edit configuration files for programs
⚠️ Notepad does not support bold text, font colors, images or tables. Use WordPad or Word for advanced formatting.

📖 Definition – Notepad

Notepad is a simple text editor used to create and edit plain text files (.txt). It is mainly used for writing notes, scripts and configuration files. It does not support advanced formatting features such as bold text, colors, images or tables. Notepad is a built-in application provided by Microsoft Windows and is useful for lightweight text editing and basic coding tasks.
